重拾数据结构与算法 笔记六 快速排序法

一、快速排序法

快速排序法的原理

QuickSort(arr, l, r) {

if(l >= r) return;

int p = partition(arr, l, r);

// 对 arr[l, p - 1] 进行排序
QuickSort(arr, l, p - 1);

// 对 arr[p + 1, r] 进行排序
QuickSort(arr, p + 1, r);

}

Partiton

循环不变量：arr[l+1…j] < v ; arr[j+1…i-1] > v

QuickSort-1

问题：对于完全有序的数组时间复杂度是O(n^2)的

解决：为快速排序法添加随机化

// 生成 [l, r] 之间的随机索引
int p = l + (new Random()).nextInt(r - l + 1);
swap(arr, l, p);
