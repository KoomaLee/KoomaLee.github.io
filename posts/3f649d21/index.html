一、二分搜索树

二叉树

class Node {
E e;
Node left; // 左孩子
Node right; // 右孩子
}

二分搜索树基础

  • 二分搜索树是二叉树

  • 二分搜索树的每个节点值大于其左子树的所有节点的值

  • 二分搜索树的每个节点值小于其右子树的所有节点的值

  • 每一颗子树也是二分搜索树

  • 存储的元素必须有可比较性
