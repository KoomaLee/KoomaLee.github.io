重拾数据结构与算法 笔记五 归并排序法

一、归并排序法

归并排序法的原理

MergeSort(arr, l, r) {

if(l >= r) return;
int mid = (l + r) / 2;

// 对 arr[l, mid] 进行排序
MergeSort(arr, l, mid);

// 对 arr[mid + 1, r] 进行排序
MergeSort(arr, mid + 1, r);

// 将 arr[l, mid] 和 arr[mid + 1, r] 合并
merge(arr, l, mid, r);

}
