SpringBoot的条件注解与配置 Conditonal条件注解与SpringBoot自动装配解析

一、Conditonal条件注解

策略模式的实现方案

1.byname 切换bean name

2.@Qualifier 指定bean

3.有选择的只注入一个bean 注释掉某个bean上的@Component注解

4.使用@Primary注解

5.使用@Conditonal条件注解

自定义条件注解

@Conditional + Condition

HeroConfiguration
IreliaCondition
DianaCondition

成品条件注解

  • @ConditionalOnProperty(value = “hero.condition”, havingValue = “diana”, matchIfMissing = true)

  • @ConditionalOnBean(name = “mysql”)

  • @ConditionalOnMissingBean(name = “mysql”)

  • …

二、SpringBoot自动装配解析

@SpringBootApplication -> @EnableAutoConfiguration -> @Import -> Selector -> spring.factories

@EnableXXX: 模块装配

LOLApplication
EnableLOLConfiguration
LOLConfigurationSelector

SPI: Service Provider Interface
